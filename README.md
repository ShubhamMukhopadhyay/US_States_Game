U.S. States Guessing Game

An interactive Python game built using the turtle graphics module where players try to guess all 50 U.S. states. Correct guesses are displayed on the map in real time, making learning geography fun and engaging.

📸 Preview

A U.S. map appears on the screen and the player types state names.
Each correct answer is placed at its actual location on the map.

✨ Features
🗺️ Interactive U.S. map interface
🎯 Real-time state name placement
📊 Tracks correct guesses
💾 Generates a states_to_learn.csv file for missed states
🐍 Built completely with Python
📚 Great beginner-friendly project for learning Turtle Graphics & Pandas

🛠️ Technologies Used
Python
Turtle Graphics
Pandas

🚀 How to Run
1️⃣ Clone the Repository
git clone https://github.com/your-username/us-states-game.git

2️⃣ Navigate to the Project Folder
cd us-states-game

3️⃣ Install Requirements
pip install pandas

4️⃣ Run the Game
python main.py

🎮 How to Play
Run the program.
Enter the name of a U.S. state in the input box.
Correct guesses will appear on the map.
Type Exit anytime to quit.
A CSV file containing missed states will be created for learning purposes.
